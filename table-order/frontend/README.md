# Table Order Frontend

Next.js 기반 테이블오더 서비스 프론트엔드

## 요구사항
- Node.js 20+

## 설정

```bash
cd frontend
npm install
```

## 환경 변수 (.env.local)

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## 실행

```bash
npm run dev
```

- 고객용: http://localhost:3000/customer/login
- 관리자용: http://localhost:3000/admin/login

## 빌드

```bash
npm run build
npm start
```
